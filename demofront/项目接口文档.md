# 企业访客小程序 - 接口文档

Base URL: `http://localhost:8080/api` | 统一请求头: `Authorization: Bearer <token>` | 统一响应: `{ code, msg, data }`

---

## 1. 认证模块

**1.1 微信登录**
```
POST /auth/login
Body: { code: "wx.login()获取的code" }
Resp: { token, user: { id, username, name, role, phone, avatar } }
```

**1.2 账号密码登录（开发用）**
```
POST /auth/login/bypass
Body: { username, password }
Resp: { token, user: { ... } }
```
预置账号: admin/admin123(管理员)  zhangsan/123456(被访人)  lisi/123456(访客)  guard/123456(门岗)

## 2. 访客模块

**2.1 提交预约申请** `POST /appointment`
```json
Body: { visitorName, visitorPhone, company, purpose, carPlate, visitorCount, remark, hostId, startTime, endTime }
Resp: { appointmentId }
```

**2.2 我的预约记录** `GET /appointment/my?page=&size=&status=`
```json
Resp: { list: [{ id, visitorName, company, hostName, purpose, startTime, endTime, status, qrCode, createTime, greeting? }], total }
```
status: `pending`待审核 `approved`已通过 `rejected`已拒绝 `cancelled`已取消 `confirmed`已核验

**2.3 撤销预约** `PUT /appointment/{id}/cancel`

**2.4 再次预约** `POST /appointment/{id}/rebook`

**2.5 获取通知列表** `GET /notification?page=&size=`
```json
Resp: { list: [{ id, title, content, createTime }], total }
```

**2.6 查看AI迎接话术** `GET /appointment/{id}/greeting`
```json
Resp: { greeting, seatSuggestion, notes }
```

**2.7 个人资料**
```
GET  /user/profile           # 获取个人资料
PUT  /user/profile           # 修改个人资料
Body: { name, phone, avatar? }
```

---

## 3. 被访人模块

**3.1 被访记录** `GET /appointment/host?page=&size=&status=&startDate=&endDate=`
```json
Resp: { list: [{ id, visitorName, company, purpose, startTime, status, phone }], total }
```

**3.2 被访统计** `GET /appointment/host/stats?startDate=&endDate=`
```json
Resp: { totalVisits, companyStats: [{ name, count }] }
```

**3.3 待审批列表** `GET /appointment/host/pending`
```json
Resp: [{ id, visitorName, company, purpose, startTime, visitorPhone, visitorCount, remark }]
```

**3.4 审批预约** `PUT /appointment/{id}/approve`
```json
Body: { status: "approved"|"rejected", remark? }
Resp: { greeting?, seatSuggestion?, notes? }  // 通过时返回AI话术
```

**3.5 辅助预约** `POST /appointment/helper`
```json
Body: { visitorName, visitorPhone, company, purpose, startTime, endTime }
```

**3.6 个人资料** `GET|PUT /user/profile`（同2.7）

## 4. 管理员模块

**4.1 员工管理**
```
GET    /admin/employee?page=&size=&keyword=&departmentId=    # 分页查询
POST   /admin/employee                                        # 新增
PUT    /admin/employee/{id}                                   # 修改
DELETE /admin/employee/{id}                                   # 删除
POST   /admin/employee/batch                                  # 批量操作 { ids, action: "delete"|"enable"|"disable" }
```

**4.2 部门管理**
```
GET    /admin/department?keyword=        # 查询列表（树形）
POST   /admin/department                 # 新增 { name, parentId, manager }
PUT    /admin/department/{id}            # 修改
DELETE /admin/department/{id}            # 删除
```

**4.3 管理员管理**
```
GET    /admin/admin?page=&size=          # 管理员列表
POST   /admin/admin                      # 新增管理员 { username, password, name, phone }
PUT    /admin/admin/{id}                 # 修改
DELETE /admin/admin/{id}                 # 删除
```

**4.4 角色权限** `PUT /admin/role/{roleId}`
```json
Body: { permissions: ["employee:view","appointment:approve",...] }
```

**4.5 系统设置**
```
GET  /admin/settings        # 获取系统配置
PUT  /admin/settings        # 更新系统配置 { companyName, logo, contactPhone, ... }
```

**4.6 节假日设置**
```
GET    /admin/holiday?year=       # 节假日列表
POST   /admin/holiday             # 新增 { name, date, type }
DELETE /admin/holiday/{id}        # 删除
```

**4.7 审核管理**
```
GET  /admin/appointment/pending?page=&size=         # 所有待审核预约
PUT  /admin/appointment/{id}/approve                 # 审核（同3.4）
```

**4.8 数据统计**
```json
GET /admin/stats/overview
Resp: { todayAppointments, pendingApprovals, totalEmployees, totalDepartments }

GET /admin/stats/visitor-record?startDate=&endDate=
Resp: { timeStats: [{ date, count }], companyStats: [{ name, count }], deptStats: [{ name, count }] }

GET /admin/stats/trend?days=7
Resp: [{ date, count }]  // 预约趋势
```

**4.9 通知管理**
```
GET    /admin/notification?page=&size=
POST   /admin/notification    Body: { title, content, targetRole? }
PUT    /admin/notification/{id}
DELETE /admin/notification/{id}
```

## 5. 门岗模块

**5.1 扫码核验** `POST /guard/verify`
```json
Body: { qrCode: "扫描二维码内容" }
Resp: { valid: true, message, appointmentId, visitorName, company, hostName, time, status }
```

**5.2 确认放行** `PUT /guard/confirm/{appointmentId}`
